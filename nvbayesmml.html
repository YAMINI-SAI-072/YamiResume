Naïve Bayes Classification

Implementation of Naïve Bayes Algorithm for Classification

Project Overview

This project demonstrates the Naïve Bayes classification technique, a probabilistic classifier based on Bayes theorem with strong (naïve) independence assumptions between features.

Features

  • Bayesian probabilistic model
  • Classification tasks
  • Handling categorical and continuous data
  • Performance evaluation metrics
