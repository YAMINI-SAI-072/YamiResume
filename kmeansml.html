K-Means Clustering

Unsupervised Clustering using K-Means Algorithm

Project Overview

This project demonstrates K-Means clustering algorithm for unsupervised learning, grouping data points into k clusters based on feature similarity.

Features

  • Unsupervised learning
  • Cluster centroids calculation
  • Iterative assignment of points
  • Visualization of clusters
